## 1. 로그인

### 접속 방법
- 제공받은 URL로 접속합니다.
- 관리자로부터 전달받은 **아이디**와 **비밀번호**를 입력합니다.
- 로그인 후 자동으로 메인 화면으로 이동합니다.

> **참고**: 5회 연속 로그인 실패 시 60초간 잠금됩니다.

## 2. 메인 화면 구성

### 사이드바 (왼쪽)
- **회사 로고 / 학교 배너**: 상단에 표시됩니다.
- **프로필 카드**: 닉네임, 역할(STUDENT/TEACHER), 소속 학교가 표시됩니다.
- **로그아웃 버튼**: 프로필 카드 아래에 위치합니다.
- **크레딧 잔액**: 현재 사용 가능한 크레딧이 표시됩니다.
- **탭 목록**: 사용 가능한 AI 기능들이 나열됩니다.

### 메인 영역 (오른쪽)
- 선택한 탭의 기능이 표시됩니다.
- 각 탭별로 프롬프트 입력, 결과 확인, 갤러리 등의 기능을 사용할 수 있습니다.

### 플로팅 채팅 (우측 하단)
- 💬 버튼을 클릭하면 실시간 채팅창이 열립니다.
- 같은 학교의 학생/교수와 대화할 수 있습니다.

## 3. 크레딧 시스템

모든 AI 기능은 **크레딧**을 소비합니다. 크레딧이 부족하면 해당 기능을 사용할 수 없으며, 관리자에게 충전을 요청해야 합니다.

### 기능별 크레딧 비용

| 기능 | 설명 | 크레딧 |
|------|------|--------|
| 💬 GPT Chat | AI 대화 1회 | 1 |
| 🎨 Image Create (MJ) | 이미지 생성 1회 (4장) | 8 |
| 🍌 NanoBanana | 이미지 생성 1회 | 5 |
| 🎬 Video Create | 영상 생성 1초당 | 7 |
| 🔊 TTS (텍스트→음성) | 음성 생성 1회 | 5 |
| 🔊 VTV (음성→음성 변환) | 음성 변환 1회 | 10 |
| 🔊 SFX (효과음) | 효과음 생성 1회 | 2 |
| 🔊 Voice Clone | 음성 복제 | 무료 |

### 크레딧 확인 방법
- 사이드바의 **크레딧 잔액** 박스에서 현재 잔액을 확인할 수 있습니다.
- 기능 사용 후 우측 상단에 "크레딧 차감 완료 (잔여: N)" 알림이 표시됩니다.

## 4. 💬 GPT Chat

OpenAI GPT 모델과 대화할 수 있는 채팅 기능입니다.

### 사용 방법
1. 왼쪽 탭 목록에서 **GPT Chat**을 선택합니다.
2. 하단 입력창에 질문이나 요청을 입력합니다.
3. Enter 또는 전송 버튼을 클릭합니다.
4. AI의 응답이 채팅 형태로 표시됩니다.

### 주요 기능
- **새 대화 시작**: 상단의 + 버튼으로 새 대화를 생성할 수 있습니다.
- **대화 이력**: 이전 대화 내용이 자동으로 저장됩니다.
- **멀티턴 대화**: 이전 맥락을 기억하여 자연스러운 대화가 가능합니다.

### 활용 예시
- 에세이 작성 도움
- 코드 작성 및 디버깅
- 번역 및 문법 교정
- 아이디어 브레인스토밍

## 5. 🎨 Image Create (Midjourney)

Midjourney AI를 이용하여 고품질 이미지를 생성합니다.

### 사용 방법
1. 왼쪽 탭 목록에서 **Image Create**을 선택합니다.
2. 프롬프트 입력창에 원하는 이미지를 영문으로 설명합니다.
3. 생성 버튼을 클릭합니다.
4. 약 30초~2분 후 4장의 이미지가 생성됩니다.

### 설정 옵션

| 설정 | 설명 |
|------|------|
| **Image Size** | Portrait(세로) / Square(정사각) / Landscape(가로) 선택 |
| **Aspect Ratio** | 슬라이더로 세부 비율 조정 (1:2 ~ 2:1) |
| **Mode** | Standard(기본) / Raw(사실적) |
| **Stylization** | 낮을수록 프롬프트에 충실, 높을수록 예술적 |
| **Weirdness** | 높을수록 독특하고 실험적인 결과 |
| **Variety** | 높을수록 4장 간 다양성 증가 |
| **Speed** | Relax 고정 |

### 프롬프트 작성 팁
- **영어**로 작성하면 더 좋은 결과를 얻을 수 있습니다.
- 구체적인 설명을 포함하세요: 스타일, 조명, 구도, 색감 등
- 예시: `A cozy coffee shop interior, warm lighting, watercolor style, soft colors`

### 이미지 첨부
- 기존 이미지를 참조하여 새 이미지를 생성할 수 있습니다.
- **Image Prompts**: 참조 이미지 기반 생성
- **Style Reference**: 스타일만 참조
- **Omni Reference**: 인물/물체 유사성 유지

## 6. 🍌 NanoBanana (이미지 생성)

Google Gemini 기반 이미지 생성 기능입니다. 멀티턴(대화형)으로 이미지를 점진적으로 수정할 수 있습니다.

### 사용 방법
1. 왼쪽 탭 목록에서 **NanoBanana**를 선택합니다.
2. 프롬프트 입력창에 원하는 이미지를 설명합니다.
3. 이미지 수와 비율을 선택합니다.
4. 생성 버튼을 클릭합니다.

### Midjourney와의 차이점

| 항목 | Midjourney | NanoBanana |
|------|-----------|------------|
| AI 모델 | Midjourney | Google Gemini |
| 언어 | 영어 권장 | 한국어 가능 |
| 이미지 수 | 4장 고정 | 1~4장 선택 가능 |
| 멀티턴 | 불가 | 이전 결과를 수정 가능 |
| 크레딧 | 8 (4장) | 5 (1회) |

### 멀티턴 편집
- 생성된 이미지를 선택하고 추가 프롬프트를 입력하면 해당 이미지를 기반으로 수정됩니다.
- 예: "배경을 바다로 바꿔줘", "색감을 더 따뜻하게"

## 7. 🎬 Video Create (영상 생성)

AI를 이용하여 텍스트 프롬프트로 영상을 생성합니다.

### 사용 방법
1. 왼쪽 탭 목록에서 **Video Create**를 선택합니다.
2. 프롬프트 입력창에 원하는 영상을 설명합니다.
3. 생성 버튼을 클릭합니다.
4. 영상 생성에는 1~5분 정도 소요됩니다.

### 주요 설정
- **영상 길이**: 생성할 영상의 길이 (초 단위)
- **크레딧**: 초당 7 크레딧이 소비됩니다.

### 프롬프트 작성 팁
- 장면의 움직임을 구체적으로 설명하세요.
- 카메라 움직임을 명시하면 더 좋은 결과를 얻을 수 있습니다.
- 예: `A golden retriever running on the beach, slow motion, sunset lighting, cinematic`

> **참고**: 영상 생성 중에는 페이지에 머물러야 합니다. 다른 탭으로 이동하면 생성이 중단될 수 있습니다.

## 8. 🔊 ElevenLabs (음성 생성)

텍스트를 자연스러운 음성으로 변환하는 기능입니다.

### 주요 기능

#### TTS (Text-to-Speech)
- 텍스트를 입력하면 자연스러운 음성으로 변환합니다.
- 다양한 목소리 중 선택할 수 있습니다.

#### VTV (Voice-to-Voice)
- 녹음한 음성을 다른 목소리로 변환합니다.
- 마이크로 직접 녹음하거나 파일을 업로드할 수 있습니다.

#### SFX (Sound Effects)
- 텍스트로 효과음을 생성합니다.
- 예: "thunder storm", "birds chirping in forest"

#### Voice Clone (음성 복제)
- 짧은 음성 샘플을 업로드하면 해당 목소리를 복제합니다.
- 복제된 목소리로 TTS를 사용할 수 있습니다.
- **무료**로 이용 가능합니다.

### 사용 방법
1. 왼쪽 탭 목록에서 **ElevenLabs**를 선택합니다.
2. 상단 탭에서 원하는 기능(TTS/VTV/SFX/Clone)을 선택합니다.
3. 텍스트 또는 음성을 입력합니다.
4. 목소리를 선택하고 생성 버튼을 클릭합니다.

## 9. 🖼️ 갤러리

같은 학교 학생들의 작품을 모아볼 수 있는 공유 갤러리입니다.

### 기능
- **이미지 갤러리**: Midjourney, NanoBanana로 생성한 이미지 모음
- **영상 갤러리**: Video Create로 생성한 영상 모음
- **음성 갤러리**: ElevenLabs로 생성한 음성/효과음 모음

### 사용 방법
1. 왼쪽 탭 목록에서 **Gallery**를 선택합니다.
2. 카테고리별로 작품을 탐색합니다.
3. 이미지를 클릭하면 새 탭에서 원본 크기로 볼 수 있습니다.

## 10. 에러 확인

작업 중 오류가 발생하면 우측 상단에 알림이 표시됩니다. 자리를 비운 사이에 발생한 에러도 확인할 수 있습니다.

### 에러 이력 확인 방법
1. 사이드바의 **"최근 에러 (N건)"** 버튼을 클릭합니다.
2. 팝업에서 발생 시각, 기능, 에러 내용을 확인할 수 있습니다.
3. 해결이 어려운 경우 플로팅 채팅으로 운영팀에 문의하세요.

## FAQ

**Q. 크레딧이 부족합니다.**
> 관리자 또는 교수님에게 크레딧 충전을 요청하세요.

**Q. 이미지/영상 생성이 오래 걸립니다.**
> Midjourney는 30초~2분, 영상 생성은 1~5분 정도 소요됩니다. 생성 중에는 페이지에 머물러 주세요.

**Q. 로그인이 안 됩니다.**
> 5회 연속 실패 시 60초 잠금됩니다. 잠시 후 다시 시도하거나, 관리자에게 문의하세요.

**Q. 생성한 작품은 어디에 저장되나요?**
> 자동으로 서버에 저장됩니다. 갤러리 탭에서 본인 및 같은 학교 학생들의 작품을 확인할 수 있습니다.

**Q. 영어로만 프롬프트를 써야 하나요?**
> Midjourney는 영어 권장, NanoBanana와 GPT는 한국어도 지원합니다.
